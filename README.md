# Сервис Мастеров (Handyman Service Platform)

Платформа для поиска и заказа услуг мастеров. Проект представляет собой веб-приложение, где клиенты могут находить профессионалов, просматривать их портфолио, оставлять отзывы, а мастера — управлять своими услугами и профилем.

## Установка и запуск

### 1. Клонирование репозитория

```bash
  git clone https://github.com/BetepIvan/web524_diplom_service_hub
```
### 2. Настройка виртуального окружения
```bash
  python -m venv venv
# Windows
  venv\Scripts\activate
# Linux/macOS
  source venv/bin/activate
```
### 3. Установка зависимостей
```bash
  pip install -r requirements.txt
```
### 4. Настройка переменных окружения
Создайте файл .env в корне проекта:
```bash
# Database
MS_SQL_USER=your_username
MS_SQL_KEY=your_password
MS_SQL_SERVER='localhost\SQLEXPRESS'
MS_SQL_DATABASE=handyman_services_db
MS_SQL_PAD_DATABASE=pad_database
MS_SQL_DRIVER=your_version  (проверено на "ODBC Driver 17 for SQL Server")

# Email
EMAIL_HOST_USER=your_email@yandex.ru
YANDEX_PASSWORD_APP=your_app_password

# Cache (optional)
CACHE_ENABLED=False
CACHE_LOCATION=redis://127.0.0.1:6379
```
### 5. Настройка базы данных
```bash
# Создание базы данных
python manage.py ccdb

# Создание миграций
python manage.py makemigrations
python manage.py migrate
```
### 6. Заполнение тестовыми данными
```bash
# Создание пользователей
python manage.py ccsu

# Загрузка фикстур (при наличии)
python manage.py loaddata fixtures/full_dump.json
```
### 7. Запуск сервера
```bash
# Создание пользователей
python manage.py runserver
```
Проект будет доступен по адресу: http://127.0.0.1:8000/

## Модели базы данных
User (Пользователь) — расширенная модель

| Поле | Тип | Описание |
|------|-----|----------|
| `email` | EmailField | Электронная почта (логин) |
| `role` | CharField | Роль: admin, moderator, master, user |
| `first_name` | CharField | Имя |
| `last_name` | CharField | Фамилия |
| `location` | CharField | Город/Адрес |
| `avatar` | ImageField | Аватар |
| `phone` | CharField | Телефон |
| `telegram` | CharField | Telegram |
| `work_days` | CharField | График работы |
| `experience` | IntegerField | Опыт (лет) |
| `education` | CharField | Образование |
| `about` | TextField | О себе |
| `service_description` | TextField | Описание деятельности |
| `portfolio` | ImageField | Портфолио |
| `categories` | ManyToManyField | Категории услуг |
| `is_master_approved` | BooleanField | Подтверждён ли мастер |

Category (Категория услуг)

| Поле | Описание |
|------|----------|
| `name` | Название категории |
| `description` | Описание |
| `icon` | Иконка FontAwesome |
| `image` | Изображение |
| `is_active` | Активна |
| `is_moderated` | Прошла модерацию |
| `is_main` | Показывать на главной |

Service (Шаблон услуги / Библиотека)

| Поле | Описание |
|------|----------|
| `title` | Название услуги |
| `category` | Категория (FK) |
| `description` | Описание |
| `is_template` | Шаблон услуги |

MasterService (Услуга мастера)

| Поле | Описание |
|------|----------|
| `master` | Мастер (FK) |
| `service_template` | Шаблон услуги (FK) |
| `price` | Цена (₽) |
| `is_active` | Активна |

Portfolio (Портфолио)

| Поле | Описание |
|------|----------|
| `master` | Мастер (FK) |
| `title` | Название работы |
| `description` | Описание |
| `image` | Фото |
| `price` | Цена работы |

Review (Отзывы)

| Поле | Описание |
|------|----------|
| `title` | Заголовок |
| `content` | Текст отзыва |
| `rating` | Оценка (1-5) |
| `author` | Автор (FK) |
| `master` | Мастер (FK) |
| `sign_of_review` | Активен |

## Основные функции
### Для клиентов (User)
🔍 Поиск и просмотр мастеров  
📍 Фильтрация по городу, опыту, рейтингу, категориям  
⭐ Оставление отзывов и оценок (1-5 звёзд)  
🚀 Возможность стать мастером
### Для мастеров (Master)
📝 Управление профилем (опыт, образование, описание)  
📚 Добавление услуг из библиотеки с индивидуальной ценой  
🖼️ Управление портфолио (примеры работ)  
👁️ Включение/выключение видимости услуг  
📊 Просмотр отзывов клиентов
### Для модераторов (Moderator)
✅ Модерация категорий (одобрение/отклонение)  
📝 Модерация отзывов (активация/деактивация/удаление)  
👥 Просмотр всех пользователей
### Для администраторов (Admin)
🛠️ Полное управление категориями  
👥 Управление пользователями  
📋 Просмотр неактивных отзывов  
🔐 Доступ к Django Admin
### Дополнительные функции
📄 Пагинация для списков мастеров, услуг, отзывов  
🔎 Поиск по категориям, услугам и мастерам  
🎛️ Фильтрация мастеров по городу, опыту, рейтингу, категориям  
⚡ Кэширование страниц через Redis  
📧 Email-уведомления при регистрации, смене пароля, добавлении услуги  
📱 Адаптивный дизайн (не полностью)

## О проекте
Данный проект разработан в рамках учебной работы по курсу Django-разработки. Проект демонстрирует навыки работы с:
- 🏗️ Кастомными моделями пользователей с ролевой системой
- 🔐 Системой прав доступа (Permission-based views)
- 🔗 Сложными отношениями между моделями (FK, ManyToMany)
- 🖼️ Работой с медиафайлами (изображения категорий, аватаров, портфолио)
- ⚡ Кэшированием через Redis
- 📧 Отправкой email-уведомлений
- 🔎 Поиском и фильтрацией с использованием Q-объектов
- ⭐ Подсчётом среднего рейтинга и отображением звёзд
- 📄 Пагинацией и вёрсткой на Bootstrap
- 🚨 Политика персональных данных
